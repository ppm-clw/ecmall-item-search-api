# DialogFlow Fulfillment Sample with Flask/python

## index
Flask使ったDialogFlowのFulfillment(WebFook)のサンプルです。

## 使い方
記述中！！！
### Pythonインストール
pythonがインストールされているかを確認  
`# python -v`
### Flaskインストール
`# pip install Flask`
### サーバ起動
`# python api.py`
### Test
ブラウザから以下にアクセスをしてJSONが表示されれば成功  
`http://localhost:5000/mall-item/api`
